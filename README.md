## About The Project

This repository hosts a modern, responsive Single Page Application (SPA) designed to serve as a professional online presence for a restaurant. Built with a focus on clean design, performance, and user experience, it effectively showcases the restaurant's menu, ambiance, and contact information.

The project demonstrates proficiency in building robust frontend applications using a contemporary technology stack.

## Live Demo

You can view the deployed application and explore its features live here:
[https://vr0cks.github.io/restaurant](https://vr0cks.github.io/restaurant)

## Features

* **Fully Responsive Design:** Optimized for seamless viewing across all devices (mobile, tablet, and desktop).
* **Modern UI/UX:** Clean, intuitive, and visually appealing user interface designed for the hospitality industry.
* **TypeScript Integration:** Ensures code quality and reliability through strong static typing.
* **Fast Development & Build:** Leverages Vite for an extremely fast development server and optimized production build process.
* **Utility-First Styling:** Styled efficiently using Tailwind CSS for rapid and scalable component styling.

## üöÄ Installation & Setup

To get a local copy of the project up and running, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/vr0cks/restaurant.git](https://github.com/vr0cks/restaurant.git) # Replace 'vr0cks/restaurant' if your repo name is different
    cd restaurant
    ```

2.  **Install project dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```
The application will typically be accessible at `http://localhost:5173` (or the port specified by Vite).
